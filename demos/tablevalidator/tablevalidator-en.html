Table Validator

  • Documentation
  • Questions or comments?

This Table Validator is an web editor tool that provide assistance to produce conforming WCAG table. This tools help to identify relationships error within a complex table provided in input. To use this validator, paste your HTML table source code in the text field below then Validate it.

Notice about using this tools

The HTML output generated by using tools may vary, but are similar, on different browser vendor. Like the element name may are rendered in upper case in Internet Explorer but not in Chrome and Firefox.

The analysis is based on the table structure provided as input. Any cells content would not interfere with any results. It is assumed that the authors provide in input table that define a well visual cell relationship and where the appropriate tabular markup is used. For further information, please consult the WCAG Guideline and the 12 techniques for tables.
