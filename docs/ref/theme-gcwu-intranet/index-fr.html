Thème de la facilité d'emploi Web GC pour les sites intranet

  • Examples pratiques
  • Questions ou commentaires?

Aperçu

Chef du projet : Chad Farquharson (@cfarquharson)

Description

Le thème de la facilité d'emploi Web GC pour les sites intranet à été créé pour conformer aux principes de la Norme sur la facilité des sites Web.

Avantages

  • Les premières tâches ont été essayées avec les utilisateurs : contactez-nous, la recherche, retourner à la page d'accueil, à propos de nous, rester connexes par les médias sociaux, naviguer le site, lire les avis, et basculer la langue officielle de choix.
  • Conformes à la Norme sur l'accessibilité des sites Web et aligner avec la Norme sur la facilité d'emploi des sites Web
  • Conformes à WCAG 2.0 AA
  • Conception réactive avec le soutien pour les appareils mobiles et les ordinateurs de table
  • Recours aux balises sémantiques HTML5 et à la spécification WAI-ARIA pour accroître l'accessibilité
  • Regroupe le gros de la navigation sous le domaine du contenu
  • Conformité aux critères HTML5 et XML

Utilisation recommandée

Fournir la couche de présentation pour les pages Web

Quand utiliser

Ce thème peut être utilisé pour construire des thèmes supplémentaires pour la Boîte à outils de l'expérience Web.

Quand ne pas utiliser

Ce thème ne devrait pas être utilisé sans modification. Il devrait être utilisé seulement comme un modèle pour un autre thème.

Apparence

Le thème adapte la BOEW pour l'usage sur les autres sites.

Contenu

Le contenu devrait se fixer sur les premières tâches des utilisateurs. Les utilisateurs ont tendance à naviguer par le zone de contenu premier et les menus deuxième. Ils cherchent des mots-clés qui les aideront à rencontrer leur besoin d'informations. Écrire dans la voix active et les verbes d'usage. Les utilisateurs scruteront souvent les informations distinctes - comme les liens. Chaque lien doit être descriptif et représentatif du contenu à lequel il mène. Ceci est surtout prononcé quand les utilisateurs accèdent au site via un appareil mobile.

Implémentation

Instructions générales

  1. Utiliser \theme-gcwu-intranet\*.html pour créer vos pages Web.
  2. Facultatif : Créer et installer un our plusieurs fichers CSS (pour le CSS personnalisé)
  3. Facultatif : Créer les liens aux nouveaux fichiers CSS entre les commentaires suivants : 
    < !-- CustomScriptsCSSStart -->
< !-- CustomScriptsCSSEnd -->
  4. Corriger les URLs des *.css, *.js et *.gif dans les fichiers *.html
  5. Remplacer le titre de la page (entre <title> et </title>,), le titre du site et le titre du contenu (entre <h1 id="wb-cont"> et </h1>)
  6. Remplacer les valeurs des métadonnées

Instructions spécifiques aux pages de contenu (cont*.html)

  1. Corriger les liens de la barre de menu ou éliminer la barre de menu
  2. Corriger les liens de sélection de la langue ou éliminer les liens de sélection de la langue
  3. Facultatif : Mettre en oeuvre navigation secondaire version 1 (trois niveaux maximum) ou navigation secondaire version 2 (quatre niveaux maximum)
  4. Corriger le titre du site
  5. Ajouter le titre du sous-site
  6. Corriger la zone de recherche ou éliminer la zone de recherche
  7. Corriger le fil d'Ariane ou éliminer le fil d'Ariane
  8. Corriger les liens dans le pied de page du site
  9. Corriger le valeur pour Date modified/Date de modification ou éliminer Date modified/Date de modification

Instructions spécifiques aux pages d'entrée (sp-pe*.html)

  1. Corriger les liens English/Français

Instructions spécifiques aux pages de messages du serveur (serv*.html)

  1. Remplacer les messages en anglais et français

Développement

Le code pour le thème de la facilité d'emploi Web GC pour les sites intranet se trouve dans plusieurs places dans le dossier source de la BOEW :

  • theme-gcwu-intranet/js/theme.js - contient le JavaScript du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/sass/theme.scss - contient le CSS pour les pages de contenu du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/sass/theme-ie.scss - contient le CSS spécifique à IE7/8 pour les pages des contenu du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/sass/theme-serv.scss - contient le CSS pour les pages de messages du serveur du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/sass/theme-serv-ie.scss - contient le CSS spécifique à IE7/8 pour les pages de messages du serveur du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/sass/theme-sp-pe.scss - contient le CSS pour les pages d'entrée du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/sass/theme-sp-pe-ie.scss - contient le CSS spécifique à IE7/8 pour les pages d'entrée du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/sass/includes/ - contient les fichiers à inclure pour créer les fichies CSS du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/jquerymobile/jquery.mobile.theme.css - contient le CSS spécifique à jQuery Mobile du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/images/ - contient les images du thème de la facilité d'emploi Web GC pour les sites intranet
  • theme-gcwu-intranet/jquerymobile/images/ - contient les images spécifique à jQuery Mobile pour le thème de la facilité d'emploi Web GC pour les sites intranet

Problèmes connus

  • Les styles des CSS ne seront pas entièrement uniformes d'un fureteur à l'autre puisque plusieurs interpréteront différemment la syntaxe des CSS.
  • L'essai de facilité d'emploi est nécessaire pour les styles dans la système styles in the CSS grid system.
